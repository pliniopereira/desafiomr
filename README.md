# desafiomr

## Used:
  * Manjaro Linux x86_64
  * Python v3.9.5

## How to run on terminal:
```
python3 comand_robot.py
```

## How to run on docker:
```
docker build -t mars_robot:latest .
docker run --rm -it mars_robot:latest
```
